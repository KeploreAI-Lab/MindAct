# 医学模型构建基础

医学模型构建通常强调数据质量、偏差控制、可解释性与合规。

## 标准流程

1. 数据治理与标注协议
2. 训练/验证/外部测试划分
3. 模型训练与超参数搜索
4. 校准、可解释性与安全评估

## 关键风险

- 数据分布偏移（医院/设备/人群差异）
- 标签噪声与金标准不一致
- 类别不平衡导致召回下降

## 常用策略

- 分层抽样与重加权
- 概率校准（Platt/Isotonic）
- 置信度阈值 + 人工复核
