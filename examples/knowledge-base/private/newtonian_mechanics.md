# 牛顿力学概要

经典力学在宏观、低速场景下仍是最常用的建模语言。

## 牛顿三定律

1. **惯性定律**：不受外力时，质点保持匀速直线运动或静止。
2. **动力学定律**：$\vec{F} = \dfrac{\mathrm{d}\vec{p}}{\mathrm{d}t}$，质量常数时即 $\vec{F} = m\vec{a}$。
3. **作用与反作用**：两物体相互作用力大小相等、方向相反、沿同一直线。

## 常见力

- **重力**：近地表 $mg$，方向竖直向下。
- **弹力**：胡克定律 $F = -kx$（一维弹簧）。
- **摩擦力**：动摩擦 $f = \mu N$；静摩擦 $f \le \mu_s N$。

## 功与能

- 功：$W = \int \vec{F}\cdot\mathrm{d}\vec{r}$。
- 动能定理：合外力做功等于动能增量。
- 保守力场存在势能；机械能守恒当且仅当非保守力不做功。
